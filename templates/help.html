Help

Useful commands for running, clearing, and debugging this project.

Run

# run the web server (default port 5000)
cargo run -- serve

# run on a custom port
cargo run -- serve --port 5001

# run in production mode flag
cargo run -- serve --prod

Build & checks

# format rust files
cargo fmt

# lint
cargo clippy -- -D warnings

# tests
cargo test

Clear / reset local pipeline artifacts

# remove generated pipeline data (be careful: destructive)
rm -rf .pipeline

# recreate state by re-running ingest/serve workflow afterwards
cargo run -- serve

Debugging tips

  • Check health endpoint: curl -s http://localhost:5000/health
  • Use browser dev tools for UI issues (console/network tabs).
  • Watch server logs in terminal while triggering actions from the UI.
  • Verify required files exist under .pipeline/ if pages are empty.
